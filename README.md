# esp32_smartwatch
DIY smartwatch. Features time(from wifi), step count, calorie count(experimental) so far. 

# Libraries used:

  TFT_eSPI.h

  SPI.h

  WiFi.h

  time.h

  Wire.h

# Hardware used:

  TTGO T-Display

  GY-521

  400mAh LiPo


Case STL here: https://www.thingiverse.com/thing:4730124
